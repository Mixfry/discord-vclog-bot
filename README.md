# vclog bot

ボイスチャット（VC）の滞在時間を記録し、滞在時間に応じて経験値を付与するDiscord Botです。

## 機能 / 仕組み
- ユーザーがVCに参加している時間を記録します。
- VCに5分間滞在するごとに1XP獲得します（退室時に付与）。
- XPが一定量貯まるとレベルが上がります。
  - 経験値テーブル: `6 + Math.floor(level / 5) * 2` 
  - 5レベルごとに次のレベルまでの必要XPが2ずつ増えていきます

## コマンド
全てのコマンドには `公開する` オプション（True/False）があり、結果をチャンネル全体に見せるか、自分だけに表示するかを選べます。

### `/mytime`
現在の自分のステータスを確認できます。
- レベル
- 獲得総XP
- 次のレベルまでの必要XP
- 合計滞在時間
- サーバー内順位

### `/ranking`
サーバー内の滞在時間ランキング（TOP 10）を表示します。
- レベル、総XP、総滞在時間の一覧を確認できます。
